# Bahubali Swami Song

```
ಬಾಹುಬಲಿ ಸ್ವಾಮಿ ಜಗಕೆಲ್ಲ ಸ್ವಾಮಿ
ಶಾಂತಿಯ ಮೂರುತಿಯೇ
ನಮಿಪೆವು ಅನುದಿನವು

ಆದಿನಾಥ ಕುವರ ಭರತನ ಸೋದರ
ಸೋದರನ ಗೆದ್ದೆಯಲ್ಲಾ
ರಾಜ್ಯವನ್ನೇ ಕೊಟ್ಟೆಯಲ್ಲಾ

ನೋಡೆ ನೀ ಕಿರಿಯವ ಆದೆ ನೀ ಹಿರಿಯವ
ವಿವೇಕ ನಿನ್ನದಾಗಿ
ತಾಳ್ಮೆಯು ಬಾಳಾಗೆ

ಶಾಂತಿಯ ವದನ ಕಾಂತಿಯ ನಿಲವು
ವಿಶ್ವಕೆ ಆದರ್ಶ
ನಿನ್ನಯ ದರುಶನವು

ಬೆಣಗೊಳ ರಾಜ ಅಗಣಿತ ತೇಜ
ಅರಳಿದ ಬಿಳಿ ಕಮಲ
ನಿನ್ನಯ ಪದ ಯುಗಲ

ಸುನಂದಾ ದೇವಿಯ ಗರ್ಭದಿ ಜನಿಸಿ
ಕೀರ್ತಿಯ ತಂದಿಹೆಯಾ
ಇಡಿಯ ಪ್ರಪಂಚದೊಳು
```

```
बाहुबली स्वामी जगकेल्ल स्वामी
शांतिय मूरुतिये
नमिपेवु अनुदिनवु

आदिनाथ कुवर भरतन सोदर
सोदरन गेद्देयल्ला
राज्यवन्ने कोट्टेयल्ला

नोडे नी किरियव आदे नी हिरियव
विवेक निन्नदागि
ताळ्मेयु बाळागे

शांतिय वदन कांतिय निलवु
विश्वके आदर्श
निन्नय दरुशनवु

बेणगोळ राज अगणित तेज
अरळिद बिळि कमल
निन्नय पद युगल

सुनंदा देविय गर्भदि जनिसि
कीर्तिय तंदिहेया
इडिय प्रपंचदोळु
```

```
Bahubali Swami jagakella swami
Shantiya moorutiye
Namipevu anudinavu

Aadinatha kuvara Bharathana sodara
Sodarana geddeyalla
Rajyavanne kotteyalla

Node nee kiriyava aade nee hiriyava
Viveka ninnadagi
Taalmeyu baalage

Shanthiya vadana kaanthiya nilavu
Vishwake aadarsha
Ninnaya darushanavu

Benagola raaja aganitha teja
Aralida bili kamala
Ninnaya pada yugala

Sunanda deviya garbhadi janisi
Keerthiya thandiheya
Idiya prapanchadolu
```
